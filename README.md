# zt-ans-bu-edb

Showroom demo lab for **Ansible Automation Platform 2.6** with **EDB Postgres Advanced Server** in an active/passive multi-datacenter DR architecture.

Demonstrates streaming replication, automated failover, and AAP activation on the standby site with RTO < 5 minutes and RPO < 5 seconds.

See [**README.adoc**](README.adoc) for full documentation.
